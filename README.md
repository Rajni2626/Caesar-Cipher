Project Description

The Caesar Cipher Converter is a simple web-based cryptography project that demonstrates the Caesar Cipher encryption and decryption technique. It allows users to securely encrypt plain text and decrypt encrypted messages by entering a custom shift value.

This project is built using HTML, CSS, and JavaScript with a clean, responsive, and user-friendly interface. It is designed for beginners to understand the basic concept of classical encryption methods.

Features

- Encrypt text using the Caesar Cipher algorithm.
- Decrypt encrypted text.
- Custom shift value support.
- Simple and responsive user interface.
- Fast and easy to use.

Technologies Used

- HTML5
- CSS3
- JavaScript

How to Use

1. Enter your text.
2. Enter the shift value.
3. Click Encrypt to encrypt the text.
4. Click Decrypt to get the original text back.

Purpose

This project was developed as a learning project to understand the fundamentals of cryptography and front-end web development.
